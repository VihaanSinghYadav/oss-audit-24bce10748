About MySQL
MySQL is a relational database management system (RDBMS) used for storing and organizing structured data. It was designed to be fast, reliable, and user-friendly, making it suitable for a wide range of applications.
One of the key reasons behind its popularity is that it is open-source. This means developers, students, and organizations can use it without facing licensing costs, making it highly accessible.
Additionally, MySQL is an essential component of the LAMP stack (Linux, Apache, MySQL, PHP), which is widely used for building dynamic web applications.

Shell Scripts Overview
As part of this project, I developed five shell scripts to demonstrate practical knowledge of Linux and open-source tools.
1. System Identity Report
This script displays essential system information such as the kernel version, current user, uptime, and overall system details. It helped me understand how to extract system-level information using shell commands.
2. FOSS Package Inspector
This script checks whether MySQL is installed on the system. It uses conditional statements to verify installation status and displays relevant details. Through this, I learned how package management works in Linux.
3. Disk and Permission Auditor
This script examines important directories, showing their disk usage and permission settings. It also checks the MySQL configuration directory. This task helped me better understand file systems and access permissions.
4. Log File Analyzer
This script reads a log file and counts how many times a specific keyword (such as “error”) appears. It gave me insight into how logs are used for monitoring systems and debugging issues.
5. Open Source Manifesto Generator
This script takes user input and generates a personalized open-source statement. It demonstrates concepts like user interaction, file handling, and string manipulation in shell scripting.
